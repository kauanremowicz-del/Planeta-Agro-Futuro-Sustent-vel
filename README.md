# 🌍 Planeta Agro: Futuro Sustentável

🚜 Tecnologia • 🌱 Sustentabilidade • 🌍 Inovação no Campo

## 🚀 Sobre o Projeto

Simulação interativa de uma fazenda inteligente que integra tecnologia, sustentabilidade e tomada de decisão agrícola em tempo real.

O sistema permite gerenciar recursos naturais, usar drones, acompanhar indicadores ambientais e evoluir uma fazenda virtual rumo à sustentabilidade.

---

## 🌱 Funcionalidades

- 🌾 Simulação de produção agrícola
- 💧 Controle de água e irrigação
- 🌳 Gestão de biodiversidade
- 🚜 Uso de tecnologia no campo
- 🛰️ Drone de monitoramento
- 🤖 IA agrícola interativa
- 🎯 Missões e objetivos
- 🏆 Ranking de desempenho
- 🏅 Certificado de sustentabilidade

---

## 💡 Objetivo

Mostrar como a tecnologia pode transformar o agronegócio, promovendo práticas sustentáveis e conscientes para o futuro do planeta.

---

## 🛠️ Tecnologias

- HTML5
- CSS3
- JavaScript
- Chart.js

---

## 🌍 Impacto

Projeto educativo focado em sustentabilidade, inovação e conscientização ambiental no setor agrícola.
